# 更新日志 (CHANGELOG)

## v0.0.3 (2026-04-05)

- **修复**: 还原 `_conf_schema.json` 中的字段名为 `pattern`，以解决 WebUI 显示不一致的问题。
- **优化**: 优化了 `/kwf list` 的逻辑，更准确地显示当前会话生效的拦截规则。
- **CI**: 修复了 GitHub Actions 自动发布工作流中的递归复制错误。

## v0.0.2 (2026-04-05)

- **修复**: 修正了 `metadata.yaml` 中的插件名称为合法的 Python 标识符（使用下划线代替连字符），解决安装失败的问题。
- **功能**: 新增 GitHub Actions 自动化发布工作流，支持通过推送 Tag 自动发布 Release。
- **优化**: 尝试优化 WebUI 条目显示（已在 v0.0.3 还原）。

## v0.0.1 (2026-04-05)

- **初始版本发布**:
  - 支持关键词、完全匹配、正则表达式三种拦截模式。
  - 支持“仅拦截引用回复”过滤条件。
  - 支持分群聊、私聊单独配置（通过 `/kwf` 指令或 WebUI）。
  - 支持全局规则设置（WebUI）。
  - 拦截仅在日志显示，不自动回复。
